# การตั้งค่าร่วมกับ LM Studio (Local AI)

LM Studio รองรับโปรโตคอล MCP ทำให้คุณสามารถเชื่อมต่อโมเดล AI ในเครื่องเข้ากับ `mcp-logger` ได้โดยตรง

## 1. ข้อกำหนดเบื้องต้น
- ติดตั้ง LM Studio รุ่นล่าสุด
- ได้รับไฟล์ไบนารี `mcp-logger` (ผ่านการคอมไพล์หรือดาวน์โหลด)

## 2. ขั้นตอนการตั้งค่า
1. เปิด LM Studio
2. ไปที่ส่วน **My Models / MCP** (หรือไอคอนปลั๊ก)
3. คลิก "Add MCP Server" หรือแก้ไขไฟล์ `mcp_config.json`
4. ป้อนการกำหนดค่าต่อไปนี้:

```json
{
  "mcpServers": {
    "mcp-logger": {
      "command": "/เส้นทาง/ไปยัง/mcp-logger",
      "args": ["run-mcp"],
      "env": {}
    }
  }
}
```

## 3. กฎเกณฑ์ที่สำคัญของ LM Studio
- **เส้นทางสัมบูรณ์**: LM Studio ต้องการเส้นทางที่สมบูรณ์ไปยังไฟล์ไบนารี
- **โหมดไม่มีโต้ตอบ**: เซิร์ฟเวอร์ MCP ต้องรันในโหมด `run-mcp` เพื่อสื่อสารผ่าน JSON-RPC โดยไม่มีการรบกวนจากข้อความทั่วไป
- **สิทธิ์การเข้าถึงเครือข่าย**: ตรวจสอบให้แน่ใจว่าไฟร์วอลล์อนุญาตให้แอพพลิเคชันส่งแพ็กเกจ UDP ไปยังที่อยู่ IP ของ Syslog

## 4. การวิเคราะห์อักขระหลายไบต์
LM Studio มักจะสามารถจัดการกับข้อมูล UTF-8 ได้ดี แต่อย่าลืมระบุ `encoding: "utf-8"` ในการเรียกใช้เครื่องมือเพื่อให้แน่ใจว่าบันทึกที่ส่งไปยัง Syslog ปลายทางนั้นถูกต้อง (โดยเฉพาะอักษรไทย)
