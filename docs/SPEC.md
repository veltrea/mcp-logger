# MCP Logger 仕様書

## 1. 目的
AI アシスタントがシステムログを生成・送信することを可能にし、AI による自律的な監視や通知、デバッグ作業を支援します。

## 2. アーキテクチャ
- **Core Logic**: `src/core_syslog.rs` (人間用と共有)。RFC 3164 ヘッダーおよび RFC 5424 MSG-UTF8 に準拠したパケット構築。
- **Transport**: MCP SDK (Rust) を使用した標準入出力経由の通信。

## 3. 共有ロジック管理
本プロジェクトは、リポジトリの独立性を保つため、コアロジックファイルを物理的にプロジェクト内で保持（コピー）します。

- 共有対象ファイル: `src/core_syslog.rs`

## 4. セキュリティ
- UDP 通信による非対面型の送信。
- 外部入力をパケットとして構築する際のサニタイズ（基本的な実装はコアロジックに準拠）。
