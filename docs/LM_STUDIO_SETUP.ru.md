# Настройка с LM Studio (Локальный ИИ)

LM Studio поддерживает протокол MCP, что позволяет подключать ваши локальные модели ИИ напрямую к `mcp-logger`.

## 1. Предварительные условия
- Установите последнюю версию LM Studio.
- Получите исполняемый файл `mcp-logger` (путем компиляции или загрузки).

## 2. Шаги настройки
1. Откройте LM Studio.
2. Перейдите в раздел **My Models / MCP** (или значок вилки).
3. Нажмите "Add MCP Server" или отредактируйте файл `mcp_config.json`.
4. Введите следующую конфигурацию:

```json
{
  "mcpServers": {
    "mcp-logger": {
      "command": "/путь/к/mcp-logger",
      "args": ["run-mcp"],
      "env": {}
    }
  }
}
```

## 3. Важные правила
- **Абсолютный путь**: LM Studio требует полный путь к исполняемому файлу.
- **Неинтерактивный режим**: Сервер MCP должен быть запущен в режиме `run-mcp` для взаимодействия через JSON-RPC без помех от обычных сообщений.
- **Сетевые разрешения**: Убедитесь, что брандмауэр позволяет приложению отправлять UDP-пакеты на IP-адрес Syslog.

## 4. Многобайтовый анализ
LM Studio обычно хорошо обрабатывает данные в кодировке UTF-8, но для обеспечения корректности логов, отправляемых в Syslog (особенно кириллицы), не забудьте указать `encoding: "utf-8"` при вызове инструмента.
