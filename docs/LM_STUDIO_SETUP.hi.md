# LM Studio (लोकल AI) के साथ सेटअप

LM Studio MCP प्रोटोकॉल का समर्थन करता है, जिससे आप अपने स्थानीय AI मॉडल को सीधे `mcp-logger` से जोड़ सकते हैं।

## 1. पूर्वापेक्षाएँ
- नवीनतम LM Studio स्थापित करें।
- `mcp-logger` बाइनरी फ़ाइल प्राप्त करें (संकलन या डाउनलोड के माध्यम से)।

## 2. सेटअप चरण
1. LM Studio खोलें।
2. **My Models / MCP** अनुभाग (या प्लग आइकन) पर जाएं।
3. "Add MCP Server" पर क्लिक करें या `mcp_config.json` फ़ाइल संपादित करें।
4. निम्नलिखित कॉन्फ़िगरेशन दर्ज करें:

```json
{
  "mcpServers": {
    "mcp-logger": {
      "command": "/पथ/से/mcp-logger",
      "args": ["run-mcp"],
      "env": {}
    }
  }
}
```

## 3. महत्वपूर्ण नियम
- **पूर्ण पथ**: LM Studio को बाइनरी फ़ाइल के पूर्ण पथ की आवश्यकता होती है।
- **गैर-इंटरैक्टिव मोड**: MCP सर्वर को बिना किसी साधारण संदेश के JSON-RPC के माध्यम से संवाद करने के लिए `run-mcp` मोड में चलना चाहिए।
- **नेटवर्क अनुमति**: सुनिश्चित करें कि फ़ायरवॉल एप्लिकेशन को Syslog IP पते पर UDP पैकेट भेजने की अनुमति देता है।

## 4. मल्टी-बाइट विश्लेषण
LM Studio आमतौर पर UTF-8 डेटा को अच्छी तरह से संभाल सकता है, लेकिन यह सुनिश्चित करने के लिए कि Syslog को भेजे गए लॉग सही हैं (विशेष रूप से हिंदी), टूल कॉल में `encoding: "utf-8"` निर्दिष्ट करना न भूलें।
